Search Cheatsheet
All of following syntax can be used in searches, in any order.
You only need to type as much of the word as necessary for it to be recognised,
e.g. "=tom" is enough to search for "=tomorrow"
 
Simple   Intervals =  
Word harry Yesterday =yesterday
Multiple words harry roof fall Today =today >today <today
Exclude words harry -roof Tomorrow =tomorrow
Phrases "harry fell off roof" Day =day =2day =10day
Or search harry | roof Week =week =2week
Simple date =24-Jul-2009 Month =month =6month
Before/after <24-jul-2010 >24-Jul-2010 Year =year <year >year
Date range 24-jul-2009..24-aug-2009 Past =past
Tags @harry Future =future
  Day, week, month, year, all accept a number before
the word (no spaces!), implying the number of intervals
 
States *   Filters #  
Todo *todo Next #next
Next *next Soon #soon
Wait *wait Wait #wait
Maybe *maybe Maybe #maybe
Done *done Todo #todo
    Done #done
Sorting / \   Due #due
Task /task \task Late #late
State /state \state    
Date /date \date    
Grouped date /gdate \gdate    
Project /project \project    
/ means search "ascending (a-z, or dates going up)
\ means "descending" (z-a, dates going down)
